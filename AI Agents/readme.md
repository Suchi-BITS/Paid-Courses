## Overview
A practical course on building autonomous reasoning systems, tool-driven agents, and multi-agent orchestration using LangGraph and modern agent-based AI architectures.

---

## Key Features
- Multi-agent collaboration
- Tool and API execution
- Agent memory (episodic, semantic, vector DB)
- Planning–execution–evaluation workflows
- Observability and safety for enterprise systems
